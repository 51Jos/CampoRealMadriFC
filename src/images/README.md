# Logo Real Madrid FC

## Instrucciones

Coloca aquí el logo de tu empresa Real Madrid FC con el nombre:

```
logo.jpeg
```

### Características del logo:
- **Nombre del archivo**: `logo.jpeg`
- **Formato**: JPEG (también soporta PNG)
- **Tamaño recomendado**: 512x512 px (o proporcional)
- **Peso máximo**: 500KB

### Ubicación final:
```
src/images/logo.jpeg
```

## Uso en la aplicación

El logo se muestra en:
- Pantalla de Login
- (Próximamente) Otras pantallas de la app

## Fallback

Si el logo no existe, la aplicación mostrará automáticamente un icono de balón de fútbol como placeholder.

---

**Nota**: Asegúrate de que el archivo se llame exactamente `logo.jpeg` para que la aplicación lo reconozca correctamente.
